# GD AutoFill

Веб-приложение для контент-менеджеров GlobalDrive: загружает `.xlsx`,
ищет подтверждённые характеристики товаров во внешних источниках, заполняет
только пустые и разрешённые поля и возвращает Excel с отчётами.

## Что умеет

- автоматически находит рабочий лист, строку заголовков и колонку товара;
- определяет одну из 10 поддерживаемых категорий;
- ищет через Serper, Bing и/или DuckDuckGo;
- всегда исключает домены из blacklist;
- читает HTML-страницы и PDF, но не обходит CAPTCHA/Cloudflare;
- сохраняет обычные cookies внутри одной обработки;
- выдерживает случайную паузу между запросами и охлаждение каждого домена;
- после CAPTCHA прекращает запросы к домену и добавляет URL для ручной проверки;
- использует Gemini только для извлечения из уже найденного текста;
- проверяет, что значение действительно присутствует в источнике;
- не меняет заполненные ячейки, UID/УИД, активность и розничную цену;
- выделяет добавленные значения жёлтым;
- создаёт листы `Отчёт`, `Проверить`, `Источники`;
- возвращает файл даже при частичном результате.
- обрабатывает длинные файлы по одному товару за цикл и сохраняет промежуточный
  Excel после каждой строки, поэтому разрыв соединения не уничтожает готовую часть;

## Локальный запуск

Требуется Python 3.11 или новее.

```bash
python -m venv .venv
# Windows:
.venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py
```

Откройте адрес, который покажет Streamlit (обычно `http://localhost:8501`).

Приложение использует встроенный HTML-парсер Python. Пакет `lxml` и
C-компилятор для установки не требуются.

## API-ключи

Приложение запускается без ключей: поиск идёт через DuckDuckGo, извлечение —
локальными строгими правилами. Для рабочего использования рекомендуется:

- `SERPER_API_KEY` — основной Google-поиск через Serper;
- `GEMINI_API_KEY` — сопоставление найденных характеристик с колонками Excel;
- `BING_API_KEY` — необязательный резервный поиск.

Скопируйте `.streamlit/secrets.toml.example` в `.streamlit/secrets.toml` и
вставьте ключи. Файл `secrets.toml` не загружайте в GitHub.

## Streamlit Community Cloud

1. Откройте [share.streamlit.io](https://share.streamlit.io).
2. Нажмите **Create app** и выберите репозиторий, ветку `main`, файл `app.py`.
   Проект фиксирует совместимую версию Python 3.12 через `runtime.txt`.
3. В **Advanced settings → Secrets** добавьте:

```toml
SERPER_API_KEY = "..."
GEMINI_API_KEY = "..."
# Необязательно:
BING_API_KEY = "..."
GEMINI_MODEL = "gemini-2.5-flash"
```

4. Нажмите **Deploy**. После сборки приложение будет доступно по постоянной ссылке.

## Работа контент-менеджера

1. Открыть ссылку приложения.
2. Для формы нового товара заменить в `B2` текст-пример `БРЕНД Модель`
   на точное название товара. Для массовой выгрузки ничего дополнительно не нужно.
3. Загрузить Excel-шаблон `.xlsx`.
4. По умолчанию выбран режим полного файла. Тестовый режим на 3 товара
   используется только для быстрой предварительной проверки.
5. Нажать **Заполнить Excel**.
6. Скачать результат и проверить листы `Проверить` и `Источники`.
7. Если качество устраивает — повторить в режиме полного файла.

## Ограничения

- Результат зависит от доступности корректных внешних источников.
- CAPTCHA и anti-bot не обходятся: источник записывается в отчёт и пропускается.
- Cookies и история браузера не подделываются; приложение использует только
  cookies, законно полученные HTTP-сессией от открытого сайта.
- Макросы `.xlsm` и старый формат `.xls` не поддерживаются.
- Для больших файлов ограничения времени Streamlit Cloud могут потребовать
  разбивать файл на части.
